Variável aleatória discreta¶

Uma varivel aleatria uma varivel cujos valores numricos so representaes de fenmenos aleatrios. Existem dois tipos de variveis aleatrias:

Uma varivel aleatria discreta uma varivel que assume valores distintos e limitados. Um exemplo uma varivel X que representa os valores de lanamento de um dado, neste cenrio X representa os valores [1, 2, 3, 4, 5, 6].

A distribuio de probabilidade de uma varivel discreta uma lista de probabilidades associada a cada possvel valor da varivel discreta.

Sendo pi a probilidade de X=xi, temos que P(X=xi) = pi, com pi tendo que respeitar duas condies:

Primeira condio: 0 < pi < 1 para cada i

Segunda condio: p1 + p2 + ... + pn = 1

Exemplo:

Variável aleatória contínua¶

Uma varivel aleatria contnua assume uma quantidade infinita de valores numricos. Por exemplo: uma varivel X que assume a altura, em centmetros, das pessoas.

A distribuio de probabilidade de uma varivel aleatria contnua uma funo de distribuio de probablidade. A probabilidade de se observar qualquer valor especfico de 0, j que a funo pode assumir valores infinitos. Portanto, a probabilidade de X estar em um conjunto de valores A representada por P(A) e pela rea abaixo da curva.

A curva que representa p(x) dada por:

Primeira condio: A curva no possui valores negativos, portanto, p(x) > 0, para todo x

Segundao condio: A rea total abaixo da curva igual a 1.

Esta curva tambm pode receber o nome de curva de densidade.

Distribuição Binomial¶

Uma distribuio onde apenas dois outputs so possveis (ex.: sucesso ou falha, perda ou ganho, vitria ou derrota, etc) e onde a probabilidade de cada opo a mesma em todas as tentativas chamada de Distribuio Binomial.

importante ressaltar que, embora as probabilidades sejam iguais, os resultados no precisam ser proporcionalmente distribudos, mesmo levando em conta a independncia de cada rodada.

Os parmetros de uma Distribuio Binomial so n que indica a quantidade de jogadas/rodadas e p que indica a probabilidade de sucesso de cada uma. A funo de probabilidade dada por:
