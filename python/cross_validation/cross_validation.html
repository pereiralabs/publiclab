Pipeline

O objeto pipeline recebe como entrada uma lista de tuplas (identificador, valor) sobre as quais executaremos operaes de transformao e classficao

Neste exemplo temos 2 transfromadores e 1 classificador:

  • Features na mesma escala
  • Reduo de dimensionalidade
  • Regresso logistica

# Definio e execuo do pipeline

                    ('pca', PCA(n_components=2)),
                    ('clf', LogisticRegression(random_state=1))])

Verificando

Mtodo estatstico univariado para selecionar as melhores features:

  • SelectKBest remove todas as features exceto as top-k com melhor score
  • f_regression estima o grau de dependencia linear entre 2 variveis aleatrias

param_range = np.logspace(-6, -1, 5)
train_scores, test_scores = validation_curve(
    SVC(), X, y, param_name="gamma", param_range=param_range,
    cv=10, scoring="accuracy", n_jobs=1)
train_scores_mean = np.mean(train_scores, axis=1)
train_scores_std = np.std(train_scores, axis=1)
test_scores_mean = np.mean(test_scores, axis=1)
test_scores_std = np.std(test_scores, axis=1)
In [31]:
plt.title("Curva de Validao com SVM")
plt.xlabel("$\gamma$")
plt.ylabel("Acurcia")
plt.ylim(0.0, 1.1)
lw = 2
plt.semilogx(param_range, train_scores_mean, label="Acurcia de treinamento",
             color="darkorange", lw=lw)
plt.fill_between(param_range, train_scores_mean - train_scores_std,
                 train_scores_mean + train_scores_std, alpha=0.2,
                 color="darkorange", lw=lw)
plt.semilogx(param_range, test_scores_mean, label="Acurcia da validao cruzada",
             color="navy", lw=lw)
plt.fill_between(param_range, test_scores_mean - test_scores_std,
                 test_scores_mean + test_scores_std, alpha=0.2,
                 color="navy", lw=lw)
plt.legend(loc="best")
plt.show()
  • O plot mostra diferentes valores para o parmetro gamma.
  • Esquerda: valores baixo para ambos --> underfitting
  • Direita: valor baixo apenas para validao, mas alto no treino --> overfitting
  • Centro: Melhores valores para gamma
